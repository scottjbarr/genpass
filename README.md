# Generate Password

A random password generator.

## Usage

    $ genpass -h
    Usage of genpass:
      -length int
            password length (default 64)

    $ genpass
    cxpTW4vaddQqiFhmM0hrhXggNHqPtNByDM8MN1pVWXzbVXu2U5Po9djTTcDnK0mU
